# plugMaintenance message

Event passed when plug goes into maintenance mode.

### Packet Example

```js
[{
    "a": "plugMaintenance",     // Event name
    "p": {},                    // Is empty
    "s": "xxxx"                 // Room name
}]
```
### Real life example
```js
[{
    "a": "plugMaintenance",
    "p": {},
    "s": "loves-kpop"
}]
```
