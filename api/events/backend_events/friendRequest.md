# friendRequest message

Event passed when you receive a friend request.


### Packet Example

```js
[{
    "a": "friendRequest",   // Event name
    "p": "xxxxxxx",         // Name of the user
    "s": "xxxx"             // Room name
}]
```
### Real life example
```js
[{
    "a": "friendRequest",
    "p": "kool_panda",
    "s": "loves-kpop"
}]
```
