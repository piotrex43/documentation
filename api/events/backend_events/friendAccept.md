# friendAccept message

Event passed when a user accepts your friend request.


### Packet Example

```js
[{
    "a": "friendAccept",    // Event name
    "p": "xxxxxx",          // Name of the user
    "s": "xxxx"             // Room name
}]
```
### Real life example
```js
[{
    "a": "friendAccept",
    "p": "kool_panda",
    "s": "loves-kpop"
}]
```
