# User Leave Event

Event passed when a user leaves the room you are in.

# Frontend

Event name: API.USER_LEAVE

### Example Listener

```js
API.on(API.USER_LEAVE, function(data){
    //This will log the packet of the event
    console.log(data);
});
```

### Packet Example

```js
{
    id: 0,
    username: '',
    avatarID: '',
    language: 'en',
    joined: undefined,
    level: 1,
    role: 0,
    gRole: 0,
    badge: undefined
}
```

# Backend

Event Name: "userLeave"

### Packet Example

```js
[{
    "a": "userLeave",       // Event name
    "p": -1,                // ID of the user
    "s": "xxxx"             // Room name
}]
```
### Real life example
```js
[{
    "a": "userLeave",
    "p": 2348789,
    "s": "loves-kpop"
}]
```
