# Unban User

Unbans a specific user from the ban list, if you have [permission](/api/roles.md). You must be manager or above to do 
this.

### Code

```js
API.moderateUnbanUser(id);
```

### Example
```
API.moderateUnbanUser(1337);
```
Unbans the user with id 1337.
