# Delete Chat

Deletes the chat message, if you have [permission](/api/roles.md).

### Code

```js
API.moderateDeleteChat(message_id);
```

### Example

```js
API.moderateDeleteChat("3779855-1509632644755");
```
Deletes message with cid "3779855-1509632644755".
